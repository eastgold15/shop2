
# 这是一个monorepo

1.安装依赖
bun  i

2. copy环境变量改成自己的

3.阿里云oss服务，需要自己弄一个oss ，把密钥填在环境变量

3. 阿里云翻译， 需要提供一个密钥

4. 启动docker compose 容器
cd  apps/backend && docker-compose up  -d

5. bun dev
启动项目
